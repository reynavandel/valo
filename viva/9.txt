2. What is Wikipedia API?

Answer:
Wikipedia API is a Python library used to access and retrieve information from Wikipedia pages programmatically.

3. Why do we use wikipediaapi?

4. What is Pydantic?

Answer:
Pydantic is a Python library used for data validation and structured data modeling.

5. Why is Pydantic used in this experiment?

Answer:
It ensures that the extracted institution details are stored in a structured and validated format.

6. What is BaseModel?

Answer:
BaseModel is the base class in Pydantic used to define structured data models.

7. What is the purpose of InstitutionDetails?

Answer:
It defines the structure for storing institution information such as founder, founded year, branches, employees, and summary.

8. What is the purpose of the fetch() (or fetch_institution_details()) function?

9. Why do we specify a User-Agent?

Answer:
Wikipedia requires a User-Agent to identify the application making the request and to follow its API usage policy.

10. What is a User-Agent?

Answer:
A User-Agent is a string that identifies the application accessing a web service.
